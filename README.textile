h1. rah_maintenance

p. "Packagist":https://packagist.org/packages/rah/rah_maintenance | "Twitter":http://twitter.com/gocom | "Donate":http://rahforum.biz/donate/rah_maintenance

p. Togglable maintenance mode option for your Preferences panel. When on, unauthenticated visitors will see your error_503 page template.

h2. Installing

p. Using "Composer":http://getcomposer.org:

bc. $ composer.phar install rah/rah_maintenance

h2. Toolshed notice

p. This is a toolshed project. Experimental and not part of the main supported product line of Rah. Not yet at least. Please use at your own risk.

h2. Requirements

p. Rah_maintenance's minimum requirements:

* Textpattern 4.6-dev
